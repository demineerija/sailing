# Старт ⛵

PWA для тренера парусного спорта. Помогает на катере во время регаты:

- **Выгодный конец стартовой линии** — пингуем PIN и СУДЬЯ, ставим ветер, видим какая сторона выгоднее, на сколько градусов и метров.
- **Перекос дистанции** — после пинга ВЕРХНЕГО буя считаем, какая сторона дистанции ближе к нему.
- **Таймер 5-4-1-0** ISAF со звуковыми сигналами.
- **График сдвигов ветра** за всю гонку.
- **История** регат и гонок, заметки.

UI на русском, крупные кнопки (≥88px), морская палитра (красный = PIN/левый борт, зелёный = СУДЬЯ/правый борт, синий = верхний знак, жёлтый = ветер).

Целевое устройство: iPhone / iPad в Safari, на катере, на солнце.

---

## Требования

- Node.js 20 или 22
- npm 10+
- Современный мобильный браузер с поддержкой Geolocation, DeviceOrientation, Wake Lock и IndexedDB. На iOS — Safari 16.4+.

## Установка

```bash
npm install
```

## Запуск dev-сервера

```bash
npm run dev
```

Откроется на `http://localhost:5173`. Для теста на телефоне в той же сети используйте URL вида `http://<ip-ноутбука>:5173`. Имейте в виду: для GPS и компаса iOS требует HTTPS (либо localhost). Для HTTPS запускайте через `vite --https` или используйте `ngrok`/`mkcert`.

## Сборка production

```bash
npm run build
```

Готовая статика — в `dist/`. Можно загрузить на любой статический хостинг (Netlify, Vercel, GitHub Pages, S3+CloudFront, локальный nginx). Главное — отдавайте по HTTPS, иначе iOS Safari заблокирует доступ к компасу и WakeLock.

Проверка локально:

```bash
npm run preview
```

## Тесты

```bash
npm test
```

Покрытие: математика стартовой линии, перекоса, расстояний, wrap-around 0/360°, санити-предупреждения.

## Установка на iPhone / iPad

1. Откройте PWA по HTTPS-адресу в Safari (например, `https://start.example.com`).
2. Тапните **Поделиться** → **На экран «Домой»** → **Добавить**.
3. Иконка «Старт» появится на рабочем столе. Запуск из неё — приложение работает в полноэкранном режиме без адресной строки.
4. При первом запуске разрешите GPS. Когда нажмёте «Указать ветер» — Safari отдельно попросит разрешить доступ к компасу.

После установки приложение работает офлайн (карта OpenStreetMap кэшируется на лету, тайлы, которые уже загружались, остаются в кэше до 30 дней).

## Как работает компас на iOS

iOS Safari требует **`DeviceOrientationEvent.requestPermission()`** из user-gesture, поэтому компас включается только когда тренер сам тапает кнопку «Указать ветер». При первом нажатии откроется системный диалог. Используется `webkitCompassHeading` — истинное направление, без поправки на магнитное склонение.

Если в настройках iOS отключена «Калибровка компаса» (Настройки → Конфиденциальность → Службы геолокации → Системные службы), Safari не сможет давать heading. В таком случае пользуйтесь ручным вводом TWD в Setup.

## Известные ограничения

- **Background**: при сворачивании приложения GPS останавливается, таймер продолжает работать через `setTimeout`, но iOS может тротлить вкладки в фоне — гудки могут запоздать на 1-2 секунды. Держите приложение на переднем плане.
- **WakeLock** на iOS работает с 16.4. На более старых версиях экран будет гаснуть; держите телефон с настройкой «Блокировка экрана = никогда» во время гонки.
- **GPS-усреднение** требует ≥3 фиксов с точностью лучше 15 м за время удержания (по умолчанию 2.5с). Если GPS плохой, кнопка покажет ошибку — не двигайтесь и попробуйте ещё.
- **Терминалом не пользуйтесь:** мокрые руки и солнце. Все важные действия — крупные тач-таргеты + long-press для re-ping.
- **Battery**: GPS + screen-on быстро сажают батарею. Держите телефон на зарядке от пауэрбанка.
- **Карта**: используются публичные тайлы OpenStreetMap (без API-ключа). При плохом интернете на воде кэшируйте акваторию заранее: откройте карту в зоне Wi-Fi, прокрутите свою акваторию во всех зумах — тайлы осядут в Service Worker кэше.

## Стек

- Vite + React 18 + TypeScript (strict)
- Tailwind CSS
- vite-plugin-pwa (injectManifest) + Workbox
- Leaflet + react-leaflet (тайлы OSM)
- Recharts (график ветра)
- idb (IndexedDB-обёртка) для хранилища
- Zustand (один глобальный стор)
- Vitest (юнит-тесты)

## Структура

```
src/
  math/sailing.ts          - вся математика (чистые функции)
  math/sailing.test.ts     - 22 теста
  store/useSailingStore.ts - zustand стор + persist в IndexedDB
  db/persistence.ts        - idb wrapper
  services/
    geolocation.ts         - watchPosition + median-усреднение для ping
    orientation.ts         - DeviceOrientation (iOS-permission flow)
    timer.ts               - 5-4-1-0 таймер
    audio.ts               - Web Audio beep
    wakeLock.ts            - не давать экрану гаснуть
  views/                   - все экраны и drawer'ы
  design/tokens.ts         - цвета и спейсинги
```
